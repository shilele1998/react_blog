## React hooks + Next.js + Egg.js 博客系统


### 系统介绍

这是一个适合写IT技术文章的博客系统，主要使用的技术是React Hooks + Next.js + Egg.js + MySql + Ant Design 。从技术上你不难看出，这个博客主要用的技术都是前端程序员熟悉的技术，所以适合前端程序员使用。

对Markdown的解析使用marked+highlight的方式，这也是目前最成熟的R解决方案。
